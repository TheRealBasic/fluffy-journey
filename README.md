# Garage Chaos Simulator

A browser management game built with Vite + React + TypeScript + Tailwind + Zustand + Framer Motion + Howler.

## Run

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Controls
- Click diagnostics/repairs to act.
- Release car to cash out.
- Call Assar once per day and Atlas once per job.
- Keyboard shortcuts: use browser find/accessibility shortcuts currently; gameplay is fully clickable.
- Save/load is automatic via localStorage (`garage-chaos-save`). Reset from menu.

## Modes
- Career (30 days)
- Endless
- Daily seeded-ish chaos

## Content
- 40 car templates
- 30 faults
- 30+ customer lines
- 20 upgrades
- 10 random events
- 8 achievements
